# Versions

## 0.1.0

Initial release of OpenGenePool DNA sequence editor.

### Features

- Linear and circular sequence visualization
- Multi-range selection with keyboard modifiers (shift, ctrl/cmd)
- Annotation layers with customizable colors
- Translation layers showing amino acid sequences
- Copy/paste support with clipboard integration
- Undo/redo history
- Zoom controls
- Mobile responsive design
- Circular mode selection behavior (gap-closing for multi-range)
- Handle drag boundaries respecting all selection segments
