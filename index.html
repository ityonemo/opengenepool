Welcome to opengenepool.

opengenepool is a web-based DNA annotation, editing and cloning tool.

To use

You will need ruby and the following rubygems:

  • sinatra
  • mysql
  • json
  • haml

    • Once you've forked to your directory of choice, type in:

    ogp.rb

    Then point your browser to:

    http://localhost:4567/

    If you'd like a more detailed guide on how to set things up, check out the README.

    There are sample genbank files provided with the package to test out its capabilities.
